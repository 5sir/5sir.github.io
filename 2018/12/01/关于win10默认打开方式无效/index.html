关于win10默认打开方式无效

发表于 2018-12-01 | 分类于 生活随笔

起因

近期win10自动更新之后，存在文件默认打开方式无法设置的情况，各大网站搜寻鲜有记录。甚至连Windows客服都是在瞎指挥！

症状

  • 修改打开方式后的文件图标不变，维持未知或者默认打开方式图标
  • 右键 → 打开方式 → 列表中选择“始终用该方式打开” 无效
  • 右键 → 属性 → 更改打开方式 → 选择相应的程序 依然无法更改

处方

  1. 文件夹 → 控制面板 → 程序和功能 → 查看已安装的更新 → 找到 KB4462919 → 右键 → 卸载更新 → 重启

  2. [2018-11-1更新] KB4462933补丁也会导致此问题，卸载补丁即可

PS： Win10系统可能有的人找不到控制面板入口 请看：传送门
