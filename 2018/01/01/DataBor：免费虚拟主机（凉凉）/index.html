免费主机介绍

注意，购物车里面是月付免费，但是它快到期会自动续费的。

配置

  • SSD：1G
  • 无限流量
  • 网站：2个
  • 子域：1个
  • 数据库：1个
  • FTP：2个
  • 邮箱：1个
  • PHP版本：5.1 / 5.2 / 5.3 / 5.4 / 5.5 / 5.6 / 7.0 / 7.1
  • PHP环境：LSAPI，Nginx + PHP-FPM，apache
  • 支持SSL
  • 防止DDOS
  • 测试地址1：http://test.5sir.cn/
  • 测试地址2：http://debug.5sir.cn/
  • 无广告

开通方法

1.先注册进入：https://databor.ru/ （注册时有Google验证，需要FQ）
2.登录后选择这款为免费主机

3.然后添加购物车，点击 Add to cart

4.点击这个红色按钮开通就可以开通了~

5.这个已经在开通中，稍等一会就变成：Service is ready to use 就可以使用了。
