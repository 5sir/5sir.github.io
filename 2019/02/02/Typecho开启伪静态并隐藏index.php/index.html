Typecho开启伪静态并隐藏index.php

发表于 2019-02-02 | 分类于 资源技巧 |
字数统计: | 阅读时长 ≈

前言

Typecho后台设置永久链接后，会在域名后加上index.php，很多人都接受不了（比如博主）。

例如如下网址：
https://5sir.cn/index.php/archives/1/

但我们希望最终的形式是这样
https://5sir.cn/1.html

那么我们如何做到这样的效果？
